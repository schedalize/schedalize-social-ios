# Schedalize Social - iOS App

iOS companion app for Schedalize Social - AI-powered social media reply generation.

## Features
- User authentication (login/signup)
- Generate AI-powered replies for social media
- Multiple tone options (friendly, professional, brief)
- Reply history
- Scheduled posts management

## Requirements
- iOS 16.0+
- Xcode 15.0+
- Swift 5.9+

## Setup
1. Open `SchedulizeSocial.xcodeproj` in Xcode
2. Select your development team
3. Build and run

## API
Backend API: Railway deployment (same as Android app)
